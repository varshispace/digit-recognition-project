# Handwritten Digit Recognition using CNN

## Project Description
This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits using the MNIST dataset.

## Features
- CNN-based model
- 98% accuracy
- Confusion matrix & graphs
- Custom image prediction

## Technologies Used
- Python
- TensorFlow
- Keras
- Google Colab

## Results
The model achieved ~98% accuracy on test data.
